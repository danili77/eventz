R33: INCLUIR UN PLUGIN

Para la realización del requisito R33: INCLUIR UN PLUGIN, tuve que decidir si hacerlo yo mismo o buscar algunos ya hechos, estuve probando varios y al final decidi incluir un reproductor de musica y aunque repetitivo un slider de imagenes en 3d.

  • R25: PASAR EVENTOS A DOCUMENTO PDF

    Para la realización del requisito opcional R25: PASAR EVENTOS A DOCUMENTO PDF, mi idea era generar el PDF del evento, para ello tuve que elegir entre la extension kartik/mpdf y la extensión mPDF elegi esta ultima por facilidad de uso,genera un PDF donde se muestra una tabla con los datos del evento en cuestion.
